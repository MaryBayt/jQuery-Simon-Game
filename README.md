# jQuery Simon Game

### Stack

* HTML5
* CSS3
* Responsive Layout
* JavaScript
* jQuery
* Git

**Description**

This is a memorization game made with jQuery: one should memorize and recall the sequence of colors and try reaching higher levels every time.
